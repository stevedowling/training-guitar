# Module 7 · Capstone: Two Setlists

> **You'll do:** perform both sets - the living-room gig with your drummer and the campfire sing-along - with the prep week, recovery rehearsals, and warm debriefs that make gig #1 the first of many, then close the loop on module 1's audit and point the system at the next twelve songs.

**Builds on:** everything - [The Campfire Set](../module-05-the-campfire-set/) and [The Family Band](../module-06-the-family-band/)

## Lessons

1. [Performance prep](./01-performance-prep.md) - the run-through week, landmark maps, the sabotage game, and nerve mechanics *(the week before each gig)*
2. [The gigs](./02-the-gigs.md) - both performances, run of show, requests, and the debrief protocol *(the events themselves)*
3. [The re-audit + what's next](./03-the-re-audit.md) - the module-1 rematch, the handover, and the next twelve songs *(one closing session)*

Take them strictly in order - prep, perform, reflect. Lesson 3 is short and should be done while the glow holds.

## When you're done

Two performances recorded and warmly debriefed; the audit file closed with evidence; a next-twelve list with one scheduled reach song; the band's next gig booked; and a practice system that no longer needs this course. That was the plan all along.
