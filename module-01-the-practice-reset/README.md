# Module 1 · The Practice Reset

> **You'll learn:** why years of playing produced fragments instead of songs, and the complete replacement system - song maps, the chunk/chain/perform method, and a practice week built around your actual life (headphones at night, noise in the day, a drummer with a bedtime).

No new guitar technique in this module - deliberately. Your hands are already ahead of your repertoire; this module fixes the system that kept them apart.

## Lessons

1. [Why you know 100 riffs and 0 songs](./01-why-riffs-arent-songs.md) - the fragment loop, the one rule that reverses it, and choosing Song One *(~15 min, no guitar)*
2. [Song anatomy](./02-song-anatomy.md) - mapping any song onto one page: sections, bar counts, joins *(~20 min + two charts)*
3. [The finishing method](./03-the-finishing-method.md) - chunk, chain, perform - and the never-stop rule *(~15 min + a session)*
4. [Your practice week](./04-your-practice-week.md) - session shapes for each slot type, and the one-line log *(~15 min + planning)*

Take them in order - each lesson's exercise feeds the next (the audit picks Song One, the chart drives the method, the method fills the week).

## When you're done

You'll have: an honest audit of your fragment pile, a charted Song One, one real session run through the finishing method (with the "before" recording safely dated), and a weekly plan with every slot typed. Then the course branches: [Module 2](../module-02-first-full-songs/) starts the electric/band track, [Module 4](../module-04-singing-while-playing/) starts the campfire/singing track - run them in parallel or binge one.
