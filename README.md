# Guitar: From Bits and Pieces to Full Songs

> **By the end of this course you'll have two real setlists you can play start to finish: an electric rock set you can perform with a beginner drummer, and a campfire acoustic set you can sing along to - plus a practice system that finishes songs instead of collecting fragments.**

Built for a player with years of experience and a hundred half-learned riffs. The problem this course attacks isn't ability - it's that nothing ever gets *finished*. Every module ends with complete, performable songs.

## 🧰 What you'll need

- **Electric guitar** (primary) and an **acoustic** for the campfire track
- **Wireless guitar headphones** - already ordered; module 1 designs your night practice around them
- **A metronome or drum-loop app** - any free one; module 3 leans on backing tracks
- **A way to record yourself** - your phone is plenty; "recorded a clean run" is this course's definition of *done*
- **A tab/chord source** - this course names songs, chords, and structures but links you to recordings and licensed tabs rather than reprinting them

## 🥁 The drummer

A 9-year-old learning drums is not a complication - she's the secret weapon. Songs were chosen so her simplest steady beat is *the actual drum part* (rock history is full of them), and module 6 is entirely about making the duo work - musically and motivationally.
